Solana Agent V3

This server is running, but the React UI is served from dist-renderer/ after a build.

One-time (or after UI changes):

npm run build:renderer
node server.js

Or use ./build-and-run.sh to build, start, and open the browser.

Development: run npm run dev:renderer in one terminal and npm start in another; use the Vite dev URL (proxies /api).

See README.md in the project root for full setup.
