# Skills: Account (legacy — Solana only)

This app uses a **Solana** wallet only. EVM/account tools are not available.

For wallet, balance, transfer, and tx history, see **`workspace/skills/solana/SKILLS.md`** and **TOOLS.md**. Native SABTC/SAETH/SAUSD: **`solana_token_balance`** with **`token_symbol`** and **`solana_agent_token_send`** (send). The app wallet is configured in Settings; do not ask the user for an address.
