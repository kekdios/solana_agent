# Skills: Account (legacy — Solana only)

This app uses a **Solana** wallet only. EVM/account tools are not available.

For wallet, balance, transfer, and tx history, see **`workspace/skills/solana/SKILLS.md`** and **TOOLS.md** (solana_address, solana_balance, solana_transfer, etc.). The app wallet is configured in Settings; do not ask the user for an address.
